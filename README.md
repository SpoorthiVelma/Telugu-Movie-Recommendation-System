# Telugu-Movie-Recommendation-System
A Java Based movie recommendation system for Telugu and Pan-India movies using content-based filtering techniques.The system allows users to search movies, filter movies based on genres, and view movie recommendations through an interactive frontend interface.
The project consists of:
1.Java-based recommendation logic
2.HTML/CSS/JavaScript frontend UI
3.CSV dataset containing movie information
Features

1.Movie Search Functionality
2.Genre-Based Filtering
3.Content-Based Movie Recommendation
4.Responsive User Interface
5.Movie Poster Visualization
6.Dynamic Movie Card Display

Technologies Used

1.Backend
  Java
2.Frontend
  HTML
  CSS
  JavaScript
3.Dataset
  CSV File Dataset
  
Java Concepts Used

  Object-Oriented Programming (OOP)
  Classes and Objects
  ArrayList
  File Handling
  Exception Handling
  String Operations
  Recommendation Technique

The project uses:

  Content-Based Filtering
  Movies are recommended based on genre similarity.
  Example:
  If a user searches for:
  RRR
  The system recommends movies with similar genres such as:
  Baahubali
  Salaar
  Magadheera

Frontend

  The frontend was designed using:
  HTML for webpage structure
  CSS for styling and responsive design
  JavaScript for search functionality and dynamic UI interactions
  
  The interface includes:
  movie cards
  search bar
  genre filters
  recommendation section
  movie posters

Backend Logic

  Java handles:
  dataset processing
  movie searching
  genre filtering
  recommendation generation
  Movie data is processed from a CSV dataset using Java file handling and collections.
